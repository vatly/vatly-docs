# One-off products

One-off products are digital products you can sell to your contacts and groups. When a contact buys a one-off product, an Order gets created and any webhooks configured through the dashboard get fired. {{ className: 'lead' }}

A one-off product is a digital product that can be bought once. {{ className: 'lead' }}

Looking for subscriptions plans? See the [Subscription plans API](/subscription-plans) instead. {{ className: 'lead' }}

## The one-off product model

Below you'll find all properties for the Vatly One-off Product API resource.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the one-off product.
  </Property>
  <Property name="name" type="string">
    The name of the one-off product.
  </Property>
  <Property name="description" type="string">
    A description of the one-off product.
  </Property>
  <Property name="price" type="number">
    The price of the one-off product.
  </Property>
  <Property name="currency" type="string">
    The currency of the one-off product price.
  </Property>
  <Property name="createdAt" type="string">
    The timestamp when the one-off product was created, in ISO 8601 format.
  </Property>
  <Property name="updatedAt" type="string">
    The timestamp when the one-off product was last updated, in ISO 8601 format.
  </Property>
</Properties>

## List all one-off products

<Row>
  <Col>

    This endpoint retrieves a list of all one-off products.

    ### Optional parameters

    <Properties>
      <Property name="limit" type="integer">
        The number of one-off products to return (default: 10, max: 100).
      </Property>
      <Property name="startingAfter" type="string">
        A cursor for use in pagination. Returns results after this product ID.
      </Property>
      <Property name="endingBefore" type="string">
        A cursor for use in pagination. Returns results before this product ID.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/one-off-products">

    ```bash
    curl -X GET https://api.vatly.com/v1/one-off-products \
      -H "Authorization: Bearer {your_api_key}"
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": [
        {
          "id": "prod_1234567890",
          "name": "Example Product",
          "description": "This is an example one-off product",
          "price": 19.99,
          "currency": "EUR",
          "createdAt": "2023-01-01T00:00:00Z",
          "updatedAt": "2023-01-01T00:00:00Z"
        },
        // ... more products
      ],
      "meta": {
        "total": 50,
        "limit": 10,
        "offset": 0
      }
    }
    ```

  </Col>
</Row>
